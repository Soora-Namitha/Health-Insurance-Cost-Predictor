# Health Insurance Cost Predictor

A Machine Learning project that predicts medical insurance costs based on user details such as age, BMI, smoking habits, number of children, gender, and region.

## Project Overview

This project uses Machine Learning algorithms to estimate health insurance charges.  
The model is trained using insurance dataset features and provides cost predictions based on user inputs.

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning model training
- Insurance cost prediction
- Performance evaluation

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset Features

- Age
- Gender
- BMI
- Number of Children
- Smoking Status
- Region
- Insurance Charges
